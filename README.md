# ThallOS

A web-based desktop OS UI built with React, TypeScript, and Vite. Features draggable, resizable windows with a Windows 98-inspired aesthetic.

## Features

- Draggable windows via title bar (double-click to maximize/restore)
- 8-directional resize handles
- Window controls: minimize, maximize, restore, close
- Windows 98-style font and styling

## Development

```bash
pnpm dev
```

## Lint

```bash
pnpm lint
pnpm lint:fix
```

## Build

```bash
pnpm build
```
